## 使用 ssh连接服务器
```sh
ssh -o ServerAliveInterval=180 -o ServerAliveCountMax=3 -o KeepAlive=yes [user]@[ip]
## 如果你是重装服务器可能还需要重置下ssh
ssh-keygen -R [ip]
```

## clone project
```sh
git clone https://github.com/uyaki/dotfiles.git /home/uyaki/dotfiles
```

> 文件默认操作路径 `/home/uyaki/dotfiles`
